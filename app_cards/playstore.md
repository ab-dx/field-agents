Google Play Store — App Instruction Card (for DroidRun Agents)
App Overview

Google Play Store is Android’s official app marketplace for:

Searching apps/games/books

Installing & updating apps

Managing subscriptions and payments

Writing reviews & ratings

Checking download status and permissions

Managing library and installed apps

Play Store UI varies slightly by device/region, but core structure stays consistent.

Core Navigation & UI Map
Bottom Navigation Tabs (most common)

Games

Apps

Search (sometimes top bar only)

Offers (optional)

Books (optional)

Top Area (important)

Search bar → fastest way to find anything

Profile icon (top-right) → Account / Manage apps / Settings / Payments

✅ Most reliable workflow: Search bar → App page → Install / Update / Open

Common Tasks & Exact Steps
1) Search for an App & Open the Correct Page

Goal: Find a specific app (e.g., “Instagram”)
Steps:

Tap the Search bar

Type app name (or exact package name if available)

Tap Search/Enter

Select the correct result from list

Confirm by checking:

App icon

Developer name

App title

✅ Tip: If multiple clones exist, use developer name as the main verification.

2) Install an App

Steps:

Open app page

Tap Install

Wait for download + install to finish

Tap Open

⚠️ If “Cancel” appears, installation is in progress. Don’t tap randomly.

3) Update an App

Method A (from app page):

Search app → Open app page

Tap Update

Method B (from Manage apps):

Tap Profile icon (top-right)

Tap Manage apps & device

Go to Updates available

Tap Update next to app (or Update all)

✅ Tip: If Update button doesn’t show, app is already up to date.

4) Uninstall an App

Steps:

Search app → Open app page

Tap Uninstall

Confirm

Alternative:
Profile → Manage apps & device → Manage → select app → Uninstall

5) Open Installed App from Play Store

Search app → Open app page

Tap Open

✅ Useful if the app icon is missing from home screen.

6) Manage Installed Apps (List View)

Steps:

Tap Profile icon

Tap Manage apps & device

Tap Manage

Scroll list or use search/filter

From here you can:

Sort by recently updated

View storage use

Uninstall multiple apps

7) Check Download / Install Progress

If an install/update is running:

You’ll see a progress circle or status text on the app page

The notification shade can also show progress (but avoid leaving Play Store unless necessary)

✅ If stuck: cancel download and retry once.

8) Write a Review / Rate an App

Steps:

Open app page

Scroll to Ratings & reviews

Tap Write a review

Choose stars + type review

Tap Post

⚠️ Some devices require app to be installed before reviewing.

9) Share an App Link

Steps:

Open app page

Tap 3-dot menu (top-right) OR Share

Select share method (copy link / message)

Frequent Popups & Interruptions (How to Handle)
“Complete account setup / Add payment method”

✅ If not needed → Skip / Not now / No thanks

“Play Protect” prompts

✅ Usually safe to allow Play Protect default behavior
⚠️ If it blocks install unexpectedly:

Confirm if the app is official and legit before proceeding

“Requires sign-in”

✅ User must sign in to Google account to install apps
Agent should not guess credentials.

Search Accuracy Rules (very important)

When selecting an app from results, verify at least TWO:

Exact app name

Developer name

Download count (optional)

“Verified by Play Protect” (optional)

✅ Never install suspicious clones.

Reliable Automation Patterns
Fast install pattern

Search → open app page → Install → Open

Update pattern

Profile → Manage apps & device → Updates available → Update all

Confirm app authenticity

Search → open page → verify developer → proceed

Scrolling & Layout Notes

App pages are long; key buttons are near the top:

Install / Update / Open

“About this app” appears mid-page

Reviews and similar apps appear lower

✅ Use controlled scrolling (small swipes).

Common Failure Modes & Fixes
Problem: “Install” button missing

Possible reasons:

Not signed in

Device incompatible

Regional restriction
Fix:

Check for “Sign in” prompt

Scroll a bit up (button sometimes pinned)

Problem: “Pending…” forever

Fix:

Wait briefly

Cancel download → retry

Check network permission popups

Problem: Wrong app installed

Fix:

Uninstall immediately

Re-search and verify developer properly

Safety Rules

Don’t install apps without explicit user intent.

Avoid tapping on ads or “Suggested for you” apps.

Don’t modify payment/subscription settings unless asked.
